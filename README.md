# naveenAutomationLabsSeleniumChallenge

## Prerequisite
### <ul>
  <li>JAVA</li>
  <li>Apache Maven</li>
  
  
## How to run
### Go to project path and open cmd.Then type ```mvn clean test```


## How to acess report
